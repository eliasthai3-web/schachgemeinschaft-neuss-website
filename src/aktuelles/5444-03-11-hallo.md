Wenn dein Repository **public** ist, ist der Code sichtbar, aber schreiben darf trotzdem nicht jeder. Schreiben können nur du und eingeladene Collaborators.

Wenn du nicht möchtest, dass Wolfgang oder Alexander mit GitHub arbeiten müssen, gäbe es später noch eine andere Lösung mit **Netlify Identity / Git Gateway**. Dann könnten Benutzer über Netlify verwaltet werden und bräuchten keinen direkten GitHub-Zugriff. Netlify beschreibt, dass Identity-Nutzer mit Git Gateway Decap-CMS-Inhalte bearbeiten können, ohne GitHub/GitLab-Zugriff auf das Repository zu
